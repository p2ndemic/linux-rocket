В утилите `wtype` (которая используется для эмуляции ввода в Wayland) для отправки сочетаний клавиш используются отдельные аргументы: сначала нажимается модификатор через флаг `-M`, затем основная клавиша, а после они отпускаются с помощью флага `-m`.

Для отправки **Super + U** выполните следующую команду в терминале:

```bash
wtype -M logo u -m logo

```

### Как это работает:

* `-M logo` — нажимает (и удерживает) клавишу **Super** (в `wtype` она называется `logo` или `win`).
* `u` — нажимает и отпускает клавишу **U**.
* `-m logo` — отпускает клавишу **Super**.

---

### Полезные альтернативные варианты

Если вам нужно сделать небольшую задержку перед нажатием (например, чтобы успеть переключиться на нужное окно), можно добавить флаг `-d` (время в миллисекундах):

```bash
wtype -d 500 -M logo u -m logo

```

Если по какой-то причине имя `logo` не срабатывает в вашем дистрибутиве, можно использовать скан-код клавиши (код левой клавиши Super обычно `125`):

```bash
wtype -M 125 u -m 125

```
